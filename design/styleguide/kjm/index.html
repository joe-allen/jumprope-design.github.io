    Element & Notes

    Example

    Panel Title

    * There is sufficient white space above the title.

    * width = 20px.

    Panel Page

    * There is white space around the blue border and title.

    * height = 56px.

    6 Columns

    * 6 column layout will be used inside 12 columns grid layout.

    * 6 column will be centered if there is only 1 column.

    Columns inside columns

    * 6 column + 6 column layout will be user if there are 2 columns.

    * 6 column + 6 column layout in large form.

    * On medium size screens and below labels with be above the input.

    Columns in 3rd screen (tercera pantalla)

    * 6 columns for the label. 6 columns for the input/field

    * 6 columns for the label. 6 columns for the input

    Titles in 3rd screen (tercera pantalla)

    * Every 3rd screen will have at least one title and blue line with data below.

    Buttons above blue lines

    * Buttons that are above the blue line should have a 5px space between each other.

    Applied Filter Dropdowns

    * Applied filters should be aligned left while the filter option icons are aligned right.

    * Applied filtered items (including delete icon) should be aligned right.

    Listas Dropdowns

    * The list dropdown is 390px in height. A scroll will be used if the 'system created list' or the 'user saved list' is more than 390px high.

    * Options to edit / delete will be shown (aligned right) on each 'user saved list' item.
